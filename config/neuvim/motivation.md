# Motivation

Neovim is fun. You can just do things that make you more *productive*. You find a new plugin that improves your workflow, and adding it is easy. It's addictive. It feels good, until at some point you realize you have a long list of plugins and can't remember all the keybinds and scripts you added months ago. Does it really help you write better code? I'm not sure. It mostly adds more complexity. That's what made me want something more minimal: a setup without much bloat, focused on only what feels necessary.
